Sobre nós

Mais que uma biblioteca. Um ecossistema.

Quem somos

Somos um time que começou um projeto open-source de automação para o WhatsApp.

O que fazemos?

O time WPPConnect cria ferramentas open-source para automatizar e integrar com o WhatsApp Web, permitindo comunicação eficiente e desenvolvimento de bots.

Nosso time

Um time de profissionais que mantém e evolui todas as soluções que oferecemos.

Por que open-source

Uma plataforma que cresce com você.

WPPConnect não é um produto. É uma fundação aberta onde desenvolvedores, agências e times constroem a próxima geração de software conversacional sobre o WhatsApp.

import { create } from '@wppconnect-team/wppconnect';

const client = await create({ session: 'main' });
client.onMessage((msg) => {
  if (msg.body === 'ping') client.sendText(msg.from, 'pong');
});

Transparente por design

Cada linha de código vive no GitHub. Audite, fork, estenda. A plataforma é da comunidade.

Testado em batalha

Move milhares de bots, fluxos de atendimento e integrações pelo mundo. Releases estáveis, breaking changes cuidadosos.

Feito com cuidado

Mantido por um time ativo que entrega, escuta e documenta. Construímos as ferramentas que nós mesmos queremos usar.

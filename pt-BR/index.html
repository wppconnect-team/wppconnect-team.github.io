Why open-source

A platform that grows with you.

WPPConnect is not a product. It's an open foundation where developers, agencies and teams build the next generation of conversational software on top of WhatsApp.

import { create } from '@wppconnect-team/wppconnect';

const client = await create({ session: 'main' });
client.onMessage((msg) => {
  if (msg.body === 'ping') client.sendText(msg.from, 'pong');
});

Transparent by design

Every line of code lives on GitHub. Audit, fork, extend. The platform belongs to the community.

Battle-tested

Powering thousands of bots, customer service flows and integrations worldwide. Stable releases, careful breaking changes.

Made with care

Maintained by an active team that ships, listens, documents. We are building tools we'd want to use ourselves.
